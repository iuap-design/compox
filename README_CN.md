##  介绍
`compox`主要包含compMgr和createApp两个部分，处理控件解析和渲染的核心机制，框架组件在此基础上进行实现



## 安装

首先选择一个目录，克隆项目文件:

```
$ git clone git@github.com:iuap-design/compox.git
```

然后进入目录安装依赖：

```
$ cd compox && npm install
```

接下来，执行下面的命令产出：

```
$ npm run product
```
