### Develop

Developers can participate in the development of compox,  but also can be based on compox two development


compox use gulp.js and webpack build the project.


clone：

```
$ git clone git@github.com:iuap-design/compox.git
```

install：

```
$ npm install
```

build：

```
$ npm run product
```
